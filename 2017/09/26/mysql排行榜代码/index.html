mysql排行榜代码

Sep 26, 2017

显示前10位名次，并显示自己名次。 1.xxx 2.xxx ….. 10.xxxx 我的当前名次: 第xx名。 这样应该如何编写，主要就是如何显示自己名次。

前十SQL：
SELECT * FROM `some_table`
ORDER BY `score`
LIMIT 10
自己名次如ls：
SELECT COUNT(`score`) FROM `some_table`
WHEREscore>(SELECT `score` FROM `some_table` WHERE id=#用户的id);
就是获得比自己分数高的人有多少
